# Análisis de ingresos de una tienda online de productos electrónicos

## Descripción del proyecto

Este proyecto analiza el desempeño comercial de una tienda online de productos electrónicos utilizando R.

El objetivo es identificar los productos, ciudades y períodos que generan mayores ingresos, mediante técnicas de limpieza, transformación, análisis y visualización de datos.

## Dataset

Se utilizó el dataset **Sales Dataset of E-commerce Electronic Products**, disponible en Kaggle.

Fuente: https://www.kaggle.com/datasets/deepanshuverma0154/sales-dataset-of-ecommerce-electronic-products

## Herramientas utilizadas

* R
* tidyverse
* dplyr
* ggplot2
* lubridate
* janitor

## Proceso de trabajo

1. Carga y exploración de datos.
2. Limpieza de registros vacíos y encabezados duplicados.
3. Conversión de variables al tipo de dato adecuado.
4. Creación de variables de negocio.
5. Análisis exploratorio de datos.
6. Generación de visualizaciones.

## Principales hallazgos

* La **MacBook Pro Laptop** fue el producto con mayor facturación, superando el millón de dólares en ingresos.
* **San Francisco** fue la ciudad con mayores ventas.
* Los meses de **abril** y **agosto** registraron los picos de facturación más altos.
